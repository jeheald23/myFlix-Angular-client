myFlix Angular App

This project is the client-side for an application based on its existing server-side code (REST API and database).

Features

  • Welcome Screen: Allows users to register and sign in.
  • Movie Card Page: Users can browse all movies in the database.
  • Additional Movie Information: Buttons to view more information about the genre, director, or synopsis of a movie.
  • Favorites List: Users can add or remove movies to their favorites list.
  • Profile Page: Users can view their information.
  • Profile Management: Buttons to update information or delete the profile.

Technologies Used

  • Angular CLI version 17.3.6
  • Angular Material version 17.3.7
  • RxJS

Installation

  1. Clone the repository.
  2. Navigate to the project directory.
  3. Run npm install to install dependencies.
  4. Run ng serve to start the development server.

Usage

  1. Open your web browser and navigate to http://localhost:4200.
  2. Register or sign in to access the application features.
  3. Explore movies, manage your profile, and enjoy!
